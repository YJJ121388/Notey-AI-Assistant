### 第二部分：痛点验证：心流割裂与收藏夹黑洞
*目的：验证心流割裂、收藏夹吃灰及检索困难等痛点。*

**Q3. 当您在放松状态下刷到一条需要深度消化的“干货”视频时，您通常会怎么做？此时心理感受如何？**
- [ ] **纠结焦虑**：想记下来但不想打断刷视频的快乐，于是匆匆收藏/截图，心里想着“以后再看”。
- [ ] **立即中断**：停下来反复观看、做笔记，哪怕打断了娱乐节奏。
- [ ] **直接划走**：虽然有价值，但现在懒得动脑子，放弃记录。
- [ ] **转发保存**：转发给文件传输助手或特定的App。

**Q4. 请回想您的抖音/小红书“收藏夹”，以下描述最符合您现状的是？**
- [ ] **吃灰仓库**：收藏了非常多，但极少回头看，甚至忘了收藏过什么。
- [ ] **临时缓存**：只在需要时（如做饭前找菜谱教程）去找，平时不看。
- [ ] **定期整理**：会定期把收藏的内容整理到专门的笔记软件中。
- [ ] **完全不看**：收藏只是为了给自己一个“我学了”的心理安慰。

**Q5. 阻碍您回顾收藏夹内容的最大原因是？（多选，按痛点程度排序）**
- [ ] **心理惯性难违**：打开抖音就是想放松，很难在那个App里切换到“学习模式”。
- [ ] **时间成本高**：视频太长，不想为了一个知识点重看一遍视频。
- [ ] **内容杂乱**：娱乐和学习内容混在一起，看着头大。
- [ ] **链接失效**：想看的时候发现视频被作者删了。

**Q6. 当您试图在收藏夹里找一个具体内容（如“川菜鱼香肉丝做法”）时，最常遇到的问题是？**
- [ ] **标题不清**：很多视频封面/标题没写清楚，得一个个点进去看。
- [ ] **定位困难**：视频很长，找不到关键步骤（如调料配比）在哪一秒。
- [ ] **搜索无力**：收藏夹内部搜索功能不好用，找不到当初那个视频。
- [ ] **没有问题**：我能很快找到。

*目的：验证 Notey 核心功能点的吸引力。*

**Q7. 【极速捕获】如果有一种方式，只需点击“分享”，无需打开新App，也无需等待，就能后台自动把视频转成文字笔记存好，您会使用吗？**
- [ ] **一定会**：这正是我急需的，不打断我刷视频太重要了。
- [ ] **可能会**：取决于保存的效果好不好。
- [ ] **不会**：我习惯截图或者直接用原App收藏。

**Q8. 【深度解析】在看知识类视频时，您认为“评论区”的内容重要吗？（如课代表总结、避雷指南、神回复）**
- [ ] **非常重要**：精华往往都在评论区，甚至比视频本身更有用。
- [ ] **比较重要**：会作为补充信息查看。
- [ ] **一般**：只看视频内容就够了。
- [ ] **不重要**：不看评论。

**Q9. 【深度解析】如果这个工具不仅能提取视频文案，还能自动把“高赞评论”和“课代表总结”一并抓取生成笔记，对您的吸引力如何？**
- [ ] **极大吸引**：这能帮我节省大量翻评论的时间。
- [ ] **有吸引力**：是个不错的加分项。
- [ ] **无感**。

**Q10. 【智能追加】当您保存了一篇“旅游攻略”笔记后，如果系统能自动去知乎/小红书搜索相关的“避雷指南”或“高赞补充”并追加到笔记后面，您怎么看？**
- [ ] **太棒了**：帮我省去了跳出App去二次搜索筛选的时间，形成完整知识库。
- [ ] **有点担心**：怕自动追加的内容不准确或太冗余。
- [ ] **不需要**：我喜欢自己搜索，不相信AI的筛选。

### 第四部分：数据主权与管理习惯
*目的：验证数据主权心理及管理/编辑需求。*

**Q11. 对于您保存下来的知识笔记，您更倾向于哪种存储方式？**
- [ ] **本地优先（数据主权）**：必须保存在我的手机/电脑本地，断网也能看，且可以导出为Markdown/PDF，不依赖特定厂商。
- [ ] **云端同步**：只要能多设备同步就行，数不数据主权无所谓。
- [ ] **无所谓**：存在哪里都行，方便看到就好。

**Q12. AI 生成的笔记可能不完美，您是否需要对笔记进行“二次编辑”和“修改”？**
- [ ] **必须支持**：我要修改错别字、删除废话、添加自己的理解（高频需求）。
- [ ] **偶尔需要**：大体对就行，偶尔改改。
- [ ] **不需要**：只当个存档，基本不改。

**Q13. 【心理补偿】当您点击“归档”后，您希望看到什么样的反馈？**
- [ ] **即时可见**：希望能在显眼的地方（如顶部栏）看到它，确认它没丢，且随时能点进去看。
- [ ] **自动消失**：存了就行，别挡视野，我要继续刷视频。
- [ ] **强弹窗提醒**：必须弹个窗告诉我“保存成功”。
